My Project

Full Frontend and Backend application "Music App".

The Frontend is made on React using redux, redux-thunk, axios, some design components are taken from MUI.

The Backend is made on Node js, express, using REST, mongoose, multer, bcrypt.

Click here to see the code

Registration/login can be accomplished by filling required fields, or throughFacebook account.

After registration/login the user can open the application and browse the content (list of artists, their albums, and tracks in those albums).

The user can also add content, and the added content will appear as unpublished only for the user who added it.

There is different interface for admin, where admin has rights to publish and delete the content.
